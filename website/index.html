Quick Guide

Welcome to ArduPilot Mission Planner - Flood Rescue GCS! Here is how to run simulations:

  • Connect: Choose a COM port at the top right, select 115200 baud, and click CONNECT.
  • Waypoints Planning: Go to the PLAN view. Click on the map to add waypoint flags. You can change altitudes or delete coordinates from the table at the bottom. Click "Write WPs" to upload waypoints to the drone.
  • Launch Drone: Go back to DATA view, click ARM VEHICLE, select AUTO mode. The Scout drone will fly the planned path.
  • Rescue survivors: When scanning, if a survivor is detected, go to the Actions tab, and click "Deploy Delivery Copter".
Reference build: Mission Planner 1.3.79.1. Simulated using standard MAVLink messages.
